# zxing

### zxing扫描二维码的库

#### 使用方法

##### 1. 加入jitpack库
```
allprojects {
    repositories {
        jcenter()
        //加入jitpack库
        maven { url 'https://jitpack.io' }

    }
    }
```


##### 2. 加入dependencies
```
compile 'com.github.WangLaoSeng:zxing:1.0'
```

##### 3. 使用二维码扫描请下载本工程,参照mainactivity代码
